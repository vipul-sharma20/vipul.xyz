# Data Assets

This directory contains data files used by various visualizations.

## Directory Structure

### `geo/`
Geographic and geospatial data files.

- **`india_state.json`** - GeoJSON file containing India state boundaries including disputed territories
  - Used by: Flight map visualizations
  - Format: GeoJSON
  - Size: ~126KB
  - Purpose: Overlays India with proper boundaries on world maps
